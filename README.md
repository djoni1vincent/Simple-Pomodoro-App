# Pomodoro App

A simple time management application built with Python and Tkinter that implements the Pomodoro Technique.

## Features

- **Work Sessions**: 25-minute focused work periods
- **Short Breaks**: 5-minute breaks between sessions
- **Long Breaks**: 15-minute breaks after every 4 work sessions
- **Visual Timer**: Real-time countdown display
- **Session Tracking**: Displays completed sessions with checkmarks

## Requirements

- Python 3.x
- Tkinter (usually included with Python)

## How to Run

```bash
python main.py
```

## How It Works

1. Click the "Start" button to begin a work session
2. The timer counts down for 25 minutes
3. After each session, a break timer starts automatically
4. Every 4 work sessions triggers a longer break
5. Click "Reset" to stop and restart
